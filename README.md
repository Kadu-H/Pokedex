# Pokedex

## Funcionamento 
Aplicação em React com paginação, localStorage e integração com api externa

## 🚀 Instalação

```
$ git clone https://github.com/viniciuspasquantonio/pokedex-reac.git
$ cd react-todolist
$ npm install
$ npm start
```
